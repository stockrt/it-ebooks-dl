h2. it-ebooks-dl

POC downloader script for "it-ebooks":http://it-ebooks.info site.

h2. Using

<pre>
<code>
    # clone
    git clone git://github.com/stockrt/it-ebooks-dl
    cd it-ebooks-dl

    # run
    ./it-ebooks-dl
    or
    ./it-ebooks-dl-selenium
</code>
</pre>
